# @donotdev/cli

Your companion CLI for getting started with the [DoNotDev Framework](https://donotdev.com) and the WAI-WAY development protocol. It helps you scaffold projects, manage deployments, and maintain code quality throughout the journey.

## Getting Started

```bash
bunx create-donotdev

OR

npm i -g @donotdev/cli@latest
```

This installs the `dndev` CLI globally and scaffolds your project. Run `dndev` to see all available commands.
